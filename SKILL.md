# Trump Psychology Analyzer
# 特朗普心理分析器 v2.0

## Skill ID
`trump-psychology-analyzer`

## 简介

通过分析特朗普的公开言论（推文、演讲、采访），解读其**意图**和**心理特征**，中英双语输出。

## 功能

1. **意图分析** ⭐ - 分析言论背后的目的和核心思路
2. **心理分析** - 识别性格特质和行为模式
3. **预测反应** - 预测对特定事件的回应

## 依赖环境

- WorkBuddy / OpenClaw 平台
- 无额外依赖

## 使用方式

### 输入
任何特朗普的公开言论（推文、演讲片段、采访内容）

### 输出格式
```markdown
## 🧠 特朗普心理分析报告 v2.0

### 📊 基础评估
| 维度 | 得分 | 说明 |

### 🎯 ⭐ 意图分析
#### 核心目的: [1句话]
#### 意图类型: ATK ★★★★☆ / DEF ★★☆☆ etc
#### 核心思路: [思维模式 + 逻辑链条]
#### 受众定位: 基本盘★★★★★

### 🔍 深度分析
[中文分析]
[English analysis]

### 📈 预测与应对
[预测反应]
```

## 意图类型

| 代码 | 类型 | 说明 |
|------|------|------|
| ATK | 攻击 | 贬低对手 |
| DEF | 防御 | 否认/扮演受害者 |
| CON | 巩固 | 强化支持者 |
| SHP | 塑形 | 定义叙事 |
| IMP | 即时反应 | 情绪发泄 |

## 思维模式

- 交易思维、零和博弈、强人叙事、受害心理、直觉至上、即时满足

## 语料来源

- Twitter/Truth Social (2009-2025)
- 竞选演讲、就职典礼
- 采访、书籍
